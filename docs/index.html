为什么写这个

"够不够用"
不是一个好问题

我是独立开发者，连续两年踩同样的坑：

账户里有钱 ≠ 自由。这个月收支平衡 ≠ 下个月也平衡。看到别人"月入十万"很焦虑，但说不清自己离"不用焦虑"到底差多远。

Freedao 把这些翻译成每天能看一眼的导航数字。不是给"早日退休"的人，是给**正在独立做事、需要下一步方向**的人。

支出分两种 投资性（预期有回报）· 消费性（必选 / 可选）。前者是资产，后者是燃料。
收入分两种 劳动收入维持跑道，被动收入通往自由。只有后者在睡觉时还能增长。
两个独立指标 Runway（能撑多久）和 Freedom Progress（走到自由的%）。不混在一起看。

第一次打开提示"已损坏"？

这是 macOS 安全机制的默认保护。两种解法，任选其一：

方法 1 · 右键打开（小白推荐）

Finder 里找到 Freedao.app → 按住 Control 右键 → "打开" → 弹窗里再点"打开"。以后双击就正常了。

方法 2 · 终端一行命令

xattr -cr /Applications/Freedao.app
